# flutter_web
 This is Image annotation tool. Written using Flutter as frontend and Python, Postgress, SQLAlchemy, RESTapis at the backend.

 Supported features:
 1. Image segmentation,
 2. Bounding boxes
 3. Keypoints

The server file is lib/flask_sql.py
